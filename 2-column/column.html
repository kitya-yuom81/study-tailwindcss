Example 2: columns-2xl (Each column ≈ 672px width)

This layout will try to fit as many columns as it can, each 672px wide.

If your screen is smaller than that, it might just be 1 column.

But on wider screens, it can create 2 or more columns.

Responsive and clean!

Example 3: columns-[300px] (Custom column width)

Custom width of 300px per column!

This gives you full control over how wide each column is.

Great for fine-tuned responsive design.

Even works with images and cards.
